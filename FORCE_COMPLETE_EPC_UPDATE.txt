========================================
FORCE COMPLETE EPC UPDATE
========================================

Run this command on your EPC to download ALL latest scripts and force a complete update:

sudo bash -c '
echo "Starting complete update..."
mkdir -p /opt/wisptools /etc/wisptools
cd /opt/wisptools
rm -f epc-checkin-agent.sh epc-snmp-discovery.sh
curl -fsSL https://hss.wisptools.io/downloads/scripts/epc-checkin-agent.sh -o epc-checkin-agent.sh
chmod +x epc-checkin-agent.sh
curl -fsSL https://hss.wisptools.io/downloads/scripts/epc-snmp-discovery.sh -o epc-snmp-discovery.sh
chmod +x epc-snmp-discovery.sh
systemctl stop wisptools-checkin 2>/dev/null
sleep 1
systemctl daemon-reload
systemctl start wisptools-checkin
sleep 2
systemctl status wisptools-checkin --no-pager | head -10
echo ""
echo "Scripts installed:"
ls -lh /opt/wisptools/*.sh
echo ""
echo "Update complete!"
'

Then immediately test with:

sudo /opt/wisptools/epc-checkin-agent.sh once
