## What is WISPTools?

WISPTools is a comprehensive LTE WISP Management Platform that provides:

- **Multi-Tenant Architecture** - Manage multiple customer organizations
- **HSS Management** - Home Subscriber Server for Open5GS
- **Device Management** - TR-069/CWMP device management via GenieACS
- **Network Planning** - Coverage mapping and PCI collision prevention
- **Monitoring** - SNMP monitoring and performance graphs
- **CBRS/SAS** - Spectrum Access System integration
- **Billing** - Customer billing and subscription management
