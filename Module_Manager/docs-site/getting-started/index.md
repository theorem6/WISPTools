# Getting Started

Welcome to WISPTools! This guide will help you get started with the LTE WISP Management Platform.

## What is WISPTools?

WISPTools is a comprehensive management platform for LTE Wireless Internet Service Providers (WISPs). It provides:

- **Multi-Tenant Architecture** - Manage multiple customer organizations
- **HSS Management** - Home Subscriber Server for Open5GS
- **Device Management** - TR-069/CWMP device management via GenieACS
- **Network Planning** - Coverage mapping and PCI collision prevention
- **Monitoring** - SNMP monitoring and performance graphs
- **CBRS/SAS** - Spectrum Access System integration

## Quick Start

### For New Users

1. **Create an Account**
   - Complete the registration wizard on the platform
   - Verify your email address

2. **Set Up Your First Tenant**
   - After registration, you'll be guided through tenant setup
   - Configure your organization details
   - Select your subscription plan

3. **Start Using the Platform**
   - Access the dashboard
   - Explore the modules
   - Begin managing your network

### For Administrators

1. **Deploy the Platform**
   - Follow the [deployment guide](/deployment/)
   - Set up backend services
   - Configure frontend

2. **Configure HSS**
   - See the [HSS Production Guide](/guides/admin-guides/hss-production)
   - Set up Open5GS
   - Connect MMEs

3. **Manage Tenants**
   - Use the [Multi-Tenant Setup Guide](/guides/admin-guides/multi-tenant-setup)
   - Configure GenieACS
   - Set up user roles

### For Developers

1. **Set Up Development Environment**
   - Clone the repository
   - Install dependencies
   - Configure environment variables

2. **Explore the Architecture**
   - Understand the [Database Structure](/guides/developer-guides/database-structure)
   - Review the [API Reference](/api/)

3. **Start Developing**
   - Check out the module structure
   - Review coding standards
   - Join the development workflow
