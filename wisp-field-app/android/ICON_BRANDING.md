# WISPTools.io App Icon Branding

## Icon Requirements

The app icon should be branded with WISPTools.io identity. Here are the specifications:

### Design Guidelines
- **Primary Color**: Blue (#60a5fa - matches web app theme)
- **Background**: Dark slate (#0f172a or #1e293b)
- **Text/Logo**: "WISPTools.io" or "WISP" with antenna/tower icon
- **Style**: Modern, professional, matches web app branding

### Icon Sizes Required

Place icons in the following directories with these sizes:

- `mipmap-mdpi/`: 48x48 px
- `mipmap-hdpi/`: 72x72 px
- `mipmap-xhdpi/`: 96x96 px
- `mipmap-xxhdpi/`: 144x144 px
- `mipmap-xxxhdpi/`: 192x192 px

### Files to Replace

Replace these files in each mipmap directory:
- `ic_launcher.png` (square icon)
- `ic_launcher_round.png` (round icon)

### Quick Icon Generation

You can use online tools like:
- [Android Asset Studio](https://romannurik.github.io/AndroidAssetStudio/icons-launcher.html)
- [App Icon Generator](https://www.appicon.co/)

Or use ImageMagick/Photoshop to create icons from a master 1024x1024px design.

### Current Icon Location

Icons are located at:
```
android/app/src/main/res/mipmap-*/
```

### After Creating Icons

1. Replace all `ic_launcher.png` and `ic_launcher_round.png` files
2. Rebuild the app: `cd android && ./gradlew assembleRelease`
3. The new icons will appear in the built APK
